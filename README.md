# Proyecto-de-Catedra-Fase-1-DPS
El proyecto consiste en el desarrollo de una aplicación móvil para la gestión y agendamiento de citas médicas, diseñada para simplificar y digitalizar el proceso de reserva de consultas para pacientes, al mismo tiempo que optimiza la gestión de la agenda para los profesionales de la salud.
